# EcoRoute  (Mazingira Logistics Platform)

A fully responsive, multi-page web application designed to optimize urban waste collection and recycling through smart predictive analytics and computer vision concepts. 

Built as a core front-end engineering assignment utilizing HTML5, CSS3, JavaScript (ES6+), and Bootstrap 5.

Project Features
- **Dynamic Routing & Optimization Analytics**: Visual mockups of predictive waste management.
- **Interactive Collection Schedules**: Live filtering by residential zone.
- **Responsive Navigation**: Seamless switching between desktop, mobile, and tablet viewports.
- **Client-Side Form Validations**: Secure communication portals for local service infrastructure.

Assignment Parameters & Milestones
- **Target Pages**: 7+ fully interlinked HTML pages
- **Minimum Target Commits**: 34+ incremental commits
- **Project Timeline**: 15 June 2026 – 28 June 2026

Tech Stack
- HTML5 / CSS3
- JavaScript (ES6+)
- Bootstrap 5 (via CDN)
